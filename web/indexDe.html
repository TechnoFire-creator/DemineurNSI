Demineur : interface console

Description : Le Démineur se présente sous forme de grille généralement composée de 18 cellules (9×9).

Certaines cases cachent une bombe (il y en a souvent 10 dans une grille de cette taille).

Le but du jeu est de dévoiler toutes les cellules libres sans faire exploser une seule bombe.

Idée : Une case avec un point d’interrogation qui, lorsqu’on clique dessus va engendrer une action positive ou négatif envers le joueur:

Action Pourcentage Description
1 Ajoute plus de mine
2 Camoufle une zone découverte
3 Change la position des mines (ce qui change aussi les chiffres)
4 Découvre une colonne ou une ligne

Cahier des charges

  • Fonction Grille :

    Fonction qui crée un dico de coords de la grille .

  • Fonction statutCaseDepart :

    fonction qui pose aléatoirement des mines et les casesSpéciales dans le dico en changeant leur status et dont le nombre de mine est proportionnelle à la taille de la grille.

  • Fonction récupererCaseStatut :

    Fonction qui récuperer le type de la case.

  • Fonction Drapeau :

    Fonction qui place un drapeau sous demande du joueur. Donc on modifie le status d'une case avec le drapeau

  • Fonction Afficher:

    Fonction qui affiche avec le numero des colonnes et les lettres des lignes.

  • Fonction MineAdjacente :

    Fonction qui indique le nombre de mine adjacente

  • Fonction DécouvrirZone :

    Fonction découvre la case sélectionnée

  • Fonction CompteNombreMine :

    fonction qui montre le compte de mine présent sur la grille, le dico

  • Fonction saisie :

    fonction qui récpére les données de l'utilisateur et la vérifie

  • Fonction main :

    fonction qui utilise les autres fonction

  • Fonction regle :

    fonction qui verifie le status de la partis Ex: gagné ou perdu

  • Fonction ActionAléatoire :

    fonction qui donne aléatoirement une fonction event

  • Fonction AjouterMine :

    fonction qui rajoute des mines dans le dico en plus de celle de départs

  • Fonction CacherZone :

    fonction qui cache aléatoire les casses découvertes

  • Fonction ChangerMine :

    fonction qui change l'emplacement des mines existants dans le dico et donc de recalculer tous les cases adjacente

  • Fonction MontrerColonne :

    fonction qui Montre un colonne alétoire

  • Fonction MontrerLigne :

    fonction qui montre une ligne

Fonctionnement du programme : fonction simple + documentation

Le but du démineur est de déminer un champ de mines sans déclencher une seule mine. Le champ de mines est une grille carrée avec des cases cachées. Chaque case peut contenir soit une mine, soit un nombre qui indique le nombre de mines adjacentes.
